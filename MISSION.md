# Lilara — MISSION

> Canonical (Level 1). Why Lilara exists. Short by design — read `VISION.md` for the full vision.

---

## The problem

Powerful AI coding agents, skills, and tool repos appear faster than anyone can vet them. People install them
**blindly**: unknown code runs with broad authority, and the user has no idea what executes or what leaves his machine.
The capability is real; the trust is missing. Today you choose between **power** (install everything, hope for the best)
and **safety** (install nothing, fall behind). That trade-off is the problem Lilara exists to delete.

## The mission

**Deliver full agent power AND full safety at the same time — never as a trade-off.**

- Collect the best capabilities, **redesign and rewrite them clean-room** (never copy), and deliver them somewhere safe
  by construction.
- Make the safe path the powerful path, so the user never has to choose.
- Be the **bounded-autonomy platform** an operator can actually trust with real authority: it acts, but only within a
  contract the user approved, and it can never cross the lines that matter.

## The co-equal law

**Productivity and security are co-equal — neither is subordinate.** *"لو مفيش انتاجيه يبقى فشل، لو مفيش امان يبقى فشل
بردو."* A version that is safe but adds friction without preventing a genuine harm has **failed**. A version that is
productive but leaks data or crosses a red line has **failed**. Both must hold, always.

## The central guarantee

1. **Data stays local** and leaves only to destinations the user approved.
2. **Every change is surfaced** for review.
3. **A strong injection guard** protects the whole system — it gates *actions*, not instructions.
4. **An inviolable core** can never be weakened — not by self-improvement, not even by the user.

## What success looks like

The user installs powerful things without fear, hands Lilara real authority, and gets more done — because the guard,
the memory, the orchestration, and the proactive heartbeat (**Breath**) work for him while the red lines hold
absolutely. Power he can trust, running where it is safe.
